Нижче — **детальний розбір**, як скласти **математичну модель задачі лінійного програмування** для мінімізації витрат на пошиття костюмів.

---

# **1. Позначимо змінні**

Нехай:

* (x_1) — кількість **чоловічих** костюмів, замовлених у **1-му ательє**
* (y_1) — кількість **жіночих** костюмів, замовлених у **1-му ательє**
* (x_2) — кількість **чоловічих** костюмів, замовлених у **2-му ательє**
* (y_2) — кількість **жіночих** костюмів, замовлених у **2-му ательє**

---

# **2. Умови виконання замовлення**

Потрібно пошити:

* 17 чоловічих костюмів:
  [
  x_1 + x_2 = 17
  ]

* 20 жіночих костюмів:
  [
  y_1 + y_2 = 20
  ]

---

# **3. Обмеження по часу роботи ательє**

### **Перше ательє**

* чоловічі: 3 шт/день
* жіночі: 2 шт/день
* фонд часу: 25 днів

Отже, загальна кількість костюмів має поміститися в робочий час:

[
\frac{x_1}{3} + \frac{y_1}{2} \le 25
]

### **Друге ательє**

* чоловічі: 4 шт/день
* жіночі: 2 шт/день
* фонд часу: 12 днів

[
\frac{x_2}{4} + \frac{y_2}{2} \le 12
]

---

# **4. Цілі функція — мінімізація витрат**

Ціни:

| Ательє | Жіночий | Чоловічий |
| ------ | ------- | --------- |
| 1      | 780     | 1100      |
| 2      | 790     | 900       |

Отже, загальна вартість:

[
Z = 1100x_1 + 780y_1 + 900x_2 + 790y_2 \rightarrow \min
]

---

# **5. Умови невід’ємності**

[
x_1, y_1, x_2, y_2 \ge 0
]

---

# **Готова математична модель**

[
\begin{aligned}
\text{Мінімізувати:}\quad
& Z = 1100x_1 + 780y_1 + 900x_2 + 790y_2 \
\text{за умов: } & \
& x_1 + x_2 = 17,\
& y_1 + y_2 = 20,\
& \frac{x_1}{3} + \frac{y_1}{2} \le 25,\
& \frac{x_2}{4} + \frac{y_2}{2} \le 12,\
& x_1, y_1, x_2, y_2 \ge 0.
\end{aligned}
]
